# SAIPOS API

Backend Node/Express com suporte a multi-tenant, webhook WhatsApp (Evolution) e roteamento de pedidos para SAIPOS ou Ana Food por tenant.

## Rodar local

```bash
npm install
cp .env.example .env
npm start
```

Aplicacao em `http://localhost:3993`.
